

Terminal App -> CLI -> Shell -> OS

## OS
1. An operating system, in charge of communication between hardware and all other software

## Shell
1. A programme that lets you interact with the operating system
2. Interpret the commands you type into CLI
3. Several different shells: Bash, zsh

## CLI
1. The concept of interacting with OS through typing commands
2. Displayed in app (ex: Terminal (macOS), Vscode )
3. Same app and same cli can run several different shells
